# 🧮 계산기 (Calculator)

## 📌 프로젝트 소개

순수 JavaScript, HTML, CSS로 만든 웹 계산기입니다. The Odin Project Foundations 코스의 마무리 프로젝트로, 그동안 학습한 자바스크립트 기초를 종합적으로 활용했습니다. 커리큘럼 지침에 따라 보안상 위험한 `eval()`과 `new Function()`은 사용하지 않았습니다.

## ✨ 기능 (Features)

- [ ] `add`, `subtract`, `multiply`, `divide` 함수
- [ ] `operate` 함수 (연산자와 두 숫자를 받아 알맞은 함수 호출)
- [ ] 숫자·연산자 버튼 UI
- [ ] 디스플레이(결과 화면)
- [ ] clear 버튼 (초기화)
- [ ] 한 번에 한 쌍만 계산 (예: 12 + 7 - 1)
- [ ] 긴 소수 결과 반올림 (디스플레이 넘침 방지)
- [ ] 0으로 나누기 처리 (에러 메시지)
- [ ] 연산자 연속 입력 처리
- [ ] 결과 표시 후 새 숫자 입력 시 새 계산 시작

### 🌟 추가 기능 (Extra Credit)

- [ ] 소수점(`.`) 입력 (중복 입력 방지)
- [ ] backspace 버튼
- [ ] 키보드 지원

## 🛠 사용 기술

- **HTML5** — 구조 및 마크업
- **CSS3** — 스타일링 및 레이아웃
- **JavaScript (ES6+)** — 계산기 로직

## 📂 파일 구조

```
calculator/
├── index.html    # 마크업 및 계산기 레이아웃
├── style.css     # 스타일
├── script.js     # 계산기 로직
└── README.md
```
